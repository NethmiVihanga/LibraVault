# LibraVault (Library Management System)

A web-based Library Management System built with HTML, CSS, JavaScript, and Firebase.

## Features

- **User Authentication:** Sign up and log in functionalities.
- **Book Management:** Add, view, and manage books in the library.
- **Member Dashboard:** Dedicated interfaces for different user roles (e.g., members).
- **Theme Support:** Includes dynamic theming (e.g., dark/light mode).
- **Firebase Integration:** Backend powered by Firebase for real-time data and authentication.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Firebase (Auth, Firestore/Database)

## Getting Started

1. Clone this repository.
2. Ensure you have your Firebase configuration set up in `firebase-config.js`.
3. Open `index.html` in your web browser to start using the application.
